## ğŸ› ï¸ Troubleshooting

### "Invalid API Key"
- Ensure it starts with `AIza`
- Check for extra spaces
- Get new key: https://aistudio.google.com/app/apikey

### "Quota Exceeded"
- Wait a few minutes
- Check your Google AI Studio quota
- Consider using different account

### "Rate Limit"
- Slow down requests
- Wait 1 minute between generations
- System has built-in rate limiting

### Chat Not Saving
- Check browser localStorage is enabled
- Clear cache if needed
- Disable private/incognito mode

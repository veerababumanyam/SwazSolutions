API REQUIREMENTS
================================================================================

Three endpoints needed (not yet implemented):

1. Contact Form Submission
   POST /api/links/{linkId}/submit-form
   - Request body: form field values
   - Response: submission confirmation
   - Integration: Email service for notifications

2. File Upload
   POST /api/profiles/me/files/{linkId}/upload
   - Request body: FormData with file
   - Response: file URL and metadata
   - Integration: Cloud storage (S3, GCS, etc.)

3. File Download
   GET /api/files/{fileId}/download
   - Query params: optional password
   - Response: file binary
   - Integration: File storage service
